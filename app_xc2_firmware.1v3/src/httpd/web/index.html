XC-2 Ethernet Card Web Server

This demonstration shows the ethernet connectivity potential of the XMOS XS1-G4 chip connected to an ethernet phy. The XS1-G4 chip drives the website, the protocol stack and the ethernet hardware through software written in C and XC.

Use the menu on the left to access features of the board.
